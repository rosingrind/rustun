# Crate stun-vectors

This crate implements [`RFC5769`](https://datatracker.ietf.org/doc/html/rfc5769). Test Vectors for Session Traversal Utilities for NAT (STUN).

Crates might include it as a development dependency for testing several parts of the STUN library.
